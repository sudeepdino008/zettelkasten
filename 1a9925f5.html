Truth and falsehood in emacs lisp

nil essentially evaluates to false, everything else is true. Note that nil has 2 meanings - empty list ( nil or () ) or false.

Conditional statements in emacs lisp

  1. When: If condition is true, evaluate all statements

(when <condition>
      <statement1>
      <statement2>
      ...
      <statementn>)

  1. and: if all are true, returns the last true value; else return nil

(and 'a 'b 'c) => c
(and 'a nil 'c) => nil

  1. if: returns the value of last executed statement

(if condition 
    then-statement
    else-statement1
    else-statement2
    ...)

  1. cond: returns the first true condition

(setq value 6)
(cond ((> value 10) "greater than 10")
      ((> value 5) "greater than 5")
      ("less than 5"))
